**Color Picker Component (React)**
This project is a simple Color Picker built using React.
It allows users to select a color using an HTML color input, and the selected color updates the background of a preview box in real time.

**Features**
- Live color preview
- Uses React useState hook
- Customizable default color
- Clean and responsive layout

  **Technologies Used**
  - React.js
  - JavaScript
  - CSS
